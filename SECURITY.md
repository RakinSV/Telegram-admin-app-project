# Security Policy

This project's full security model — what's protected, what's deliberately
out of scope, and why — is documented in the wiki:

- 🇬🇧 [Security (English)](https://github.com/RakinSV/Telegram-admin-app-project/wiki/Security)
- 🇷🇺 [Безопасность (Русский)](https://github.com/RakinSV/Telegram-admin-app-project/wiki/Security-RU)

## Reporting a vulnerability

Please open a [GitHub issue](https://github.com/RakinSV/Telegram-admin-app-project/issues)
describing the concern. For anything sensitive enough that you'd rather not
post publicly first, mention that in the issue title and details can follow
privately.

## Supported versions

This project doesn't maintain multiple release branches — only the latest
commit on `main` receives fixes. See [Updating](https://github.com/RakinSV/Telegram-admin-app-project/wiki/Updating)
for how to stay current.
